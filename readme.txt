le code source ici présent est un générateur de nuage de mots sur python version 3.10.0

les librairie python utiliser :
pip :
https://pip.pypa.io/en/stable/installation/

1.moviepy
https://zulko.github.io/moviepy/install.html
pip install moviepy

2. SpeechRecognition
https://pypi.org/project/SpeechRecognition/
pip install SpeechRecognition

3. wordcloud
https://pypi.org/project/wordcloud/
pip install wordcloud
4. matplotlib
https://matplotlib.org/stable/users/installing/index.html
python -m pip install -U matplotlib
5. Tkinter
https://riptutorial.com/tkinter/example/3206/installation-or-setup
sudo apt-get install python3-tk

comme c'est baser sur la reconnaissance vocal de la base de donne google donc il est conseiller d'être connecter sur internet pour la creation de niage de mots.
